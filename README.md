## 🇸🇦 وصف المشروع بالعربية

مشروع **Aura® Powder** هو ابتكار ريادي مستدام في مجال مستحضرات النظافة الشخصية، يعتمد على تقنية **"بودرة تتحول إلى سائل" (Powder-to-Liquid)**. يهدف المشروع إلى التخلص من شحن المياه غير الضروري في مستحضرات الاستحمام التقليدية (حيث تشكل المياه 80% من وزن الشاور جل التقليدي)، وتقليل النفايات البلاستيكية بنسبة 90% عبر شحن المسحوق الفعال جافاً داخل أكياس ورقية قابلة للتحلل العضوي، ليقوم المستخدم بخلطها بالماء في المنزل.

### الأثر البيئي والاقتصادي:
* **تقليص وزن الشحن:** ينخفض وزن شحن المنتج للوحدة من 250 جرام للشاور جل التقليدي إلى **15 جرام فقط** لشاور جل Aura®، مما يقلل البصمة الكربونية للنقل بنسبة 80%.
* **حفظ ذاتي جاف:** غياب المياه في الحالة الجافة يمنع النمو البكتيري، مما يعطي المنتج صلاحية تخزين تصل إلى **سنتين** دون الحاجة لمواد حافظة كيميائية صناعية.
* **توفير البلاستيك:** يتم الاستغناء بالكامل عن العبوات البلاستيكية أحادية الاستخدام واستبدالها بمغلفات ورقية قابلة للتحلل.

## 🇬🇧 English Description

**Aura® Powder** is a pioneering development in sustainable personal care, designed to address two critical environmental challenges:
1. **Unnecessary Water Shipping:** Conventional body washes consist of 80% to 90% water. Shipping this heavy water cargo increases transport fuel consumption and carbon emissions.
2. **Single-Use Plastic Waste:** Millions of plastic bottles end up in oceans and landfills.

By completely removing water from manufacturing and packaging the dry active formula in lightweight, biodegradable paper sachets, shipping weight is reduced by 80% (from 250g to **only 15g** per unit). Users reconstitute the premium shower gel at home by adding warm tap water.

## 🧪 التركيب العلمي / Scientific Formulation

The compound is carefully balanced biologically and chemically to ensure washing performance and dermatological safety:
* **Texapon Powder (تكسابون بودرة):** The primary active foaming and cleansing surfactant derived from natural sources, allowing instant lather in waterless formats.
* **Sodium Bicarbonate (بيكربونات الصوديوم):** Acts as a pH buffer to match natural skin levels (pH 5.5) and serves as an effervescent agent to help dissolve the powder.
* **Natural Corn Starch (نشا ذرة):** A natural skin softener and anti-caking agent that keeps the dry powder stable under high humidity.
* **Organic Essential Oils:** Premium natural scents without synthetic sensitizing allergens:
  - **Musk Pomegranate** (Golden Glow / مسك رمان).
  - **Japanese Cherry Blossom** (Baby Blue Breeze / جابانيز).
  - **Libre Scent** (Violet Velvet / ليبر).

## 🚀 الميزات المتقدمة / Features

* **Bilingual Switcher (EN/AR)**: Complete support for English and Arabic with layout directionality (`dir="rtl"` vs `dir="ltr"`).
* **Cinematic Video Hero**: A looping cinematic video background for a premium user experience.
* **Ingredients Glossary**: Detailed scientific descriptions of the biological and chemical elements of the formula.
* **Formulation Batch Calculator**: Calculates the exact chemical weights of ingredients required for any target bottle volume (100ml to 2000ml).
* **Eco-Impact Dashboard**: An interactive calculator measuring carbon reduction, plastic bottle prevention, and shipping weight savings.
* **Reconstitution Simulator**: Interactive timeline showing the hydration and gelation process stages.
* **Technical Downloads Center**: Allows direct downloads of the thesis abstract, Material Safety Datasheet (MSDS), specs sheet, and full presentation PDF.

## 🛠 التشغيل المحلي / Local Installation

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mna80455-debug/-Aura-basmala-mohammed-.git
   cd -Aura-basmala-mohammed-
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
4. **Build the production bundle**:
   ```bash
   npm run build
   ```
