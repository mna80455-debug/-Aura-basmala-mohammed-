## 🌿 About the Project

**Aura® Powder** is a sustainable personal care innovation that transforms traditional shower gel into a **lightweight dry powder**. Users simply add water at home to create their own premium shower gel.

### Why It Matters:
- 🚛 **80% lighter shipping** — eliminates unnecessary water transport
- ♻️ **Zero single-use plastic** — biodegradable paper packaging only
- 🧪 **No chemical preservatives** — dry formula is naturally shelf-stable
- 🌍 **Lower carbon footprint** — fewer trucks, less fuel, cleaner air
- 🌸 **Three premium natural scent variants** for a luxurious experience

## 🛠 Getting Started

Make sure you have [Node.js](https://nodejs.org/) (v18+) installed.

```bash
# 1. Clone the repository
git clone https://github.com/mna80455-debug/-Aura-basmala-mohammed-.git
cd -Aura-basmala-mohammed-

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev

# 4. Build for production
npm run build
```

## 📱 PWA Support

This website is a **Progressive Web App (PWA)** — users can install it directly from their browser:

| Platform | How to Install |
|---|---|
| **Chrome / Edge (Desktop)** | Click the install icon in the address bar, or the golden "Install App" button in the navbar |
| **Chrome (Android)** | Tap "Add to Home Screen" from the browser menu |
| **Safari (iOS)** | Tap Share → "Add to Home Screen" |

Once installed, the app runs in standalone mode (no browser chrome), works offline for cached pages, and provides app-like shortcuts.
